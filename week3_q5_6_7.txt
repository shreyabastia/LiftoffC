7. What is the size of a C structure? How the size of a C structure is calculated?

ans: Size of C structure is the total bytes of all elements and padding as per requirement.Structure is a user friendly datatype in C language which allows us to combine data of different types together.The size of C structure is the sum of size of individual element and padding required for proper alignment.Size of the element is determined using size of operator.
